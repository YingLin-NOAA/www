Plotting QPF Verification Scores Using METviewer

Databases loaded:
  • mv_ylin_pcp_erly (loaded at vday+2)
  • mv_ylin_pcp (loaded at vday+9, includes 6h FSS scores)
New cloud-based AWS METviewer server: https://metviewer.nws.noaa.gov

Old prototype METviewer server: http://metviewerdev.ncep.noaa.gov/metviewer1.jsp

Currently the above servers are only accessible from within the NCEP firewall.

For both servers, databases are loaded daily by an unattended cron job. mv_ylin_pcp is loaded for daym9 and includes 6h FSS. mv_ylin_pcp_erly is loaded for daym2 and does not include 6h FSS.
